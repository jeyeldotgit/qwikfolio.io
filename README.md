# QwikFolio.io

QwikFolio.io is a form-based portfolio and resume builder designed for speed.  
It allows developers, students, and professionals to generate a career-ready portfolio and printable resume in minutesâ€”without design friction.

## Key Features

- Schema-driven portfolio builder
- Clean dashboard with clear next actions
- Resume-style printable preview
- Mocked analytics (views, downloads, status)

## Tech Stack

- React + TypeScript (Vite)
- Tailwind CSS + shadcn-style UI
- Zod (schema-driven validation)

## Project Status

Frontend MVP complete.  
Backend, authentication, and persistence are intentionally out of scope for the current version.
